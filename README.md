## 📌 Project Overview
This project explores the application of Edge AI (TinyML) for the Non-Destructive Testing (NDT) of agricultural produce. The objective is to classify the ripeness of coconuts by analyzing the high-frequency acoustic resonance decay generated by a mechanical impulse. 

To achieve ultra-low latency and deploy the system on severely resource-constrained microcontrollers (e.g., ESP32 with <400KB SRAM), this study benchmarked five distinct machine learning architectures, analyzing the trade-offs between predictive accuracy, memory footprint, and inference speed.

## 🧠 Methodology & Dataset
The system relies on univariate time-series data captured by a piezoelectric transducer immediately following a mechanical strike. 
* **The Physics:** Solid, water-dense matter (unripe) yields a sharp, high-amplitude impact spike with immediate vibration decay. Hollow, mature matter (ripe) yields a lower impact spike but a prolonged acoustic decay (resonance).
* **The Challenge:** Capturing this decay requires a rapid sampling rate, generating a 30-timestep array per mechanical strike.

## 📐 Engineering Trade-offs & Conclusion
While recurrent networks (LSTMs) and convolutional networks (1D-CNN) are traditionally favored for time-series decay analysis, their hardware cost is significant. The LSTM required a persistent hidden state and complex gate logic, resulting in a Flash footprint 17x larger than the CNN and the highest inference latency.

**Optimal Edge Selection:** Despite a marginal accuracy drop of ~1.4% compared to the LSTM, the **Random Forest** proved to be the optimal architecture for this specific hardware constraint. 
* **Zero Tensor Arena:** Using `micromlgen`, the Random Forest was exported entirely as native compiled C++ `if/else` logic.
* **Interpreter Bypass:** This bypassed the TensorFlow Lite runtime entirely, driving SRAM usage to near-zero.
* **Scale Invariance:** The tree-based architecture eliminated the need for float-based normalization (MinMaxScaler) on the microcontroller, allowing the direct ingestion of raw 12-bit ADC integers, drastically reducing CPU cycles.
